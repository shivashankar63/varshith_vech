About the System

This GPS-GSM based bus tracking system addresses the inefficiencies in traditional bus services by providing real-time location data, automated ETA calculations, and a unified platform for commuters and drivers. Built with modern web technologies and designed for scalability, it reduces waiting times, improves fleet management, and enhances the overall commuting experience.

Key Technologies: GPS/A-GPS, GPRS/GSM, Android Architecture, Leaflet Maps, Real-Time Data Streaming
